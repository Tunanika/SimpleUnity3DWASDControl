# Simple Unity3D WASD Control

Simple WASD control and bounce for Unity3D.
<br>
W -> Forward<br>
A -> Left<br>
S -> Back<br>
D -> Right<br>
<br>
Space -> Jump<br>
How to use:<br>
Hierarchy > your object > Inspector<br>
Drag it into the character or object you want to play.<br>
